# Flight Fare Prediction using Machine Learning

## Project Overview

This project predicts airline ticket prices based on factors such as airline, source city, destination city, departure time, class, duration, and days left before departure.

The goal is to build a regression model that can estimate flight ticket prices accurately using historical flight data.

---

## Problem Statement

Airline ticket prices change frequently depending on multiple factors. Predicting ticket prices helps travelers make better booking decisions.

This project uses machine learning algorithms to predict flight fares.

---

## Dataset

The dataset contains flight booking information including:

- Airline
- Source City
- Destination City
- Departure Time
- Arrival Time
- Number of Stops
- Flight Duration
- Days Left before departure
- Ticket Price
